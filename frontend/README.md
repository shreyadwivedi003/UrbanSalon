# PuneGlow

PuneGlow is a modern home salon services marketplace focused on Pune, Maharashtra.

## Features

- Home salon service booking
- Beauty professional listings
- Search and filtering
- AI Beauty Assistant
- Mobile responsive design
- Customer reviews and ratings
- Partner registration portal

## Tech Stack

- Next.js
- React
- Tailwind CSS
- Framer Motion
- Vercel Deployment
